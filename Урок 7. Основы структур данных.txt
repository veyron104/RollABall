Задание № 1
Расписать плюсы и минусы изученных структур данных и расписать примеры в которых нужно будет использовать тот или иной контейнер

ArrayList
- можно добавлять разные типы данных

LinkedList
- более экономичный (по памяти) при изменении количества элементов

HashSet
- содержит только уникальные элементы

ObservableCollection
- есть событие, которое вызывается при изменении элементов или их количества

Dictionary
- обращение по ключам, допускается любой тип ключа
- он быстрее работает чем другие коллекции?

Stack
- добавляются и удаляются последние элементы

Queue
- добавляются элементы только в конец
- удаляются элементы только из начала
